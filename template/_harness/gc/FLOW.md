# GC Flow

## Stage 1 — Review

Identify stale, duplicated, low-value, or superseded content.

## Stage 2 — Classify

Decide whether each item should be:

- removed
- merged
- rewritten
- promoted to long-term memory

## Stage 3 — Apply

Update the relevant files with the smallest useful change.

## Stage 4 — Summarize

Record what was cleaned, what was preserved, and why.
