# WORKFLOW

## Standard agent usage flow
1. Read `AGENTS.md`.
2. Classify the task.
3. Use `_harness/ROUTING.md` to select the minimum relevant file.
4. Read the selected instruction file(s).
5. Execute the task.

## Typical patterns
### Repo orientation
- Read `README.md`

### Structure lookup
- Read `CATALOG.md`

### Rule/boundary check
- Read `RULES.md`

### Procedural execution
- Read `WORKFLOW.md`
- Combine with `RULES.md` if the task has constraints
