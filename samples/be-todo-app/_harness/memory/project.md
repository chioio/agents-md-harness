# Project Memory

## 长期上下文

- 这是一个与 `fe-todo-app` 呼应的基于 Golang 的 TODO App 后端项目。
- 文档语言使用中文。
- 该项目支持产品、架构、开发、测试多角色 agent 协作。
- 默认执行方式是先设计 / 对齐，再实现。
- 默认采用测试驱动开发思路。

## 稳定记录项

长期记忆主要记录：

- 产品决策
- 架构决策
- 失败案例
- 测试结论
- 接口约定

## 更新规则

- 任务开始写短期记忆
- 任务结束写短期记忆
- 用户 review 后更新短期记忆
- 稳定信息再沉淀到长期记忆
